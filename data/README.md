# Dossier pour vos documents PDF

Placez ici vos fichiers PDF de documentation technique.

## Exemples de sources publiques :

### Documentation électronique
- [Arduino Documentation](https://docs.arduino.cc/) - Téléchargez les PDFs de référence
- [Raspberry Pi Documentation](https://www.raspberrypi.com/documentation/) - Guides hardware/software

### Manuels industriels
- [Siemens Industry Support](https://support.industry.siemens.com/) - Manuels d'équipements
- [Schneider Electric Resources](https://www.se.com/ww/en/work/support/) - Documentation produits
- [ABB Library](https://library.abb.com/) - Documentation technique

### Standards & Normes
- ISO, IEC standards (si vous y avez accès)
- Documentation open-source d'équipements industriels

## Formats supportés

- ✅ PDF (.pdf)
- ❌ Word/Excel (non supporté dans cette version)

## Recommandations

- Préférez des documents **textuels** (pas des scans d'images)
- Taille recommandée : 1-50 pages par document
- Évitez les PDFs protégés par mot de passe
